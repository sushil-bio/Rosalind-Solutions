# 🧬 Rosalind Bioinformatics Solutions

## Overview
This repository contains my pure-Python programmatic solutions to the [Rosalind Bioinformatics Stronghold](http://rosalind.info/) track. These scripts demonstrate foundational skills in parsing raw genomic data, applying biological mathematics, and building efficient logic from scratch without relying on high-level data science frameworks.

## Concepts Mastered
- **Genomic String Parsing:** Transcribing DNA to RNA, generating reverse complements.
- **Data Structures:** Utilizing dictionaries (hash maps) to build and query custom amino acid codon translation tables.
- **Algorithm Design:** Sliding window iterations for motif finding and calculating Hamming Distances to find genetic mutations.
- **Mathematical Modeling:** Simulating biological population dynamics (e.g., Mortal Fibonacci Rabbits).

## Tech Stack
- **Language:** Python
- **Focus:** Raw Algorithmic Logic, Core Computer Science Fundamentals
